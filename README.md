#  BehaviorIQ  
*A Behavioral Productivity Intelligence Dashboard*

---

## 📌 Overview

**BehaviorIQ** is an interactive data science dashboard built to analyze digital productivity patterns and generate behavioral insights using machine learning.

This project combines:
- 📊 Data visualization  
- 🤖 Behavioral modeling  
- 📈 Interactive analytics  
- 🎨 Clean, modern UI design  

The goal is not just to show charts — but to provide meaningful insights from user behavior data.

---

## 🎯 Project Objectives

- Analyze digital session behavior
- Identify productivity trends
- Train a behavioral classification model
- Evaluate model performance
- Present actionable insights alongside visualizations

---

## 🤖 Machine Learning Model

The behavioral model:

- Uses features such as:
  - `duration_minutes`
  - `hour`
- Predicts productivity classification
- Evaluated using:
  - Accuracy Score
  - Confusion Matrix
  - Classification Report

---

## 📊 Key Insights

- Session duration has the strongest impact on productivity prediction.
- Time-of-day contributes moderately to behavior trends.
- The model captures structured usage patterns effectively.
- Feature importance reveals dominant behavioral signals.
